# 📻 Ràdio Personal (PWA)

Un reproductor de ràdio online lleuger, elegant i instal·lable com a aplicació nativa (PWA).

## ✨ Característiques
* **Instal·lable:** Funciona com una aplicació independent gràcies a la tecnologia PWA.
* **Emissores precarregades:** Catalunya Ràdio, RAC1, iCat, RNE, BBC, France Inter, NPR i més.
* **Emissores personalitzades:** Afegeix i elimina les teves pròpies emissores. Es guarden automàticament.
* **Animació d'ones:** Indicador visual quan l'emissora és en directe.
* **Interfície Minimalista:** Disseny coherent amb el lector EPUB i el reproductor de vídeo.
* **Tema fosc i clar.**

## 🛠️ Tecnologies utilitzades
* **HTML5/CSS3:** Estructura i disseny.
* **JavaScript:** Lògica del reproductor i Service Worker.
* **Web Audio API:** Reproducció nativa de streams MP3/AAC.
* **GitHub Pages:** Allotjament gratuït i segur.
